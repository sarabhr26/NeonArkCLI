# NeonArkCLI


Java Console Application developed for COSC 4301 under Professor Jon-Mikel Pearson.

## Description
This project simulates a Neon Ark Admin Console using:
- CSV data handling
- Input validation
- Simulated REST API requests
- JSON payload output

## Features
1. Add New Warden
2. View Wardens
3. Update Warden
4. Manage Certifications
5. Deactivate / Terminate Warden
6. Exit Menu

## Technologies Used
- Java
- IntelliJ IDEA
- CSV File Handling
